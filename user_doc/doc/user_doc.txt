 *  \section cfu-intro Introduction
 *
 *  crystal_facet_uml is a uml diagram drawing tool
 *  that creates a set of consistent uml diagrams.
 *
 *  \image html screenshot_1.png
 *  \image latex screenshot_1.png "" width=15cm
 *
 *  \section cfu-tools Tool Bar
 *
 *  \image html resources/file_use_db.png
 *  \image latex resources/file_use_db.png "" width=3cm
 *  \li Opens an existing database file or creates a new database file.
 *
 *  \image html resources/file_export.png
 *  \image latex resources/file_export.png "" width=3cm
 *  \li Exports all diagrams to the selected folder. Supported formats are txt, png and svg.
 *
 *  \image html resources/file_new_window.png
 *  \image latex resources/file_new_window.png "" width=3cm
 *  \li Opens another window on the same database.
 *
 *  \image html resources/tool_create_diagram.png
 *  \image latex resources/tool_create_diagram.png "" width=3cm
 *  \li Create a child-diagram by clicking at any space in the current diagram
 *
 *  \image html resources/tool_navigate.png
 *  \image latex resources/tool_navigate.png "" width=3cm
 *  \li Switch the current diagram to the parent diagram or to a child diagram.
 *
 *  \image html resources/tool_edit.png
 *  \image latex resources/tool_edit.png "" width=3cm
 *  \li Move classifiers in the diagram
 *  \li Mark the diagram, classifiers and relationships with pink corners
 *  \li Select the diagram, a classifier, a feature or a relationship with yellow corners.
 *  \li Edit the properties of the yellow-cornered object at the bottom of the window.
 *
 *  \image html resources/tool_create_object.png
 *  \image latex resources/tool_create_object.png "" width=3cm
 *  \li Create a classifier by clicking at an empty space in the diagram
 *  \li Create a relationship by clicking on the source classifier and dragging to the destination classifier
 *  \li Create a feature by clicking on the classifier
 *
 *  \image html resources/edit_cut.png
 *  \image latex resources/edit_cut.png "" width=3cm
 *  \li Cut all pink-cornered elements to the clipboard (features and relationships not yet supported)
 *
 *  \image html resources/edit_copy.png
 *  \image latex resources/edit_copy.png "" width=3cm
 *  \li Copy all pink-cornered elements to the clipboard (features and relationships not yet supported)
 *
 *  \image html resources/edit_paste.png
 *  \image latex resources/edit_paste.png "" width=3cm
 *  \li Pastes diagrams and classifiers from the clipboard to the uml model.
 *      If id and name are identical to an existing element, an instance of the existing element is pasted to the diagram.
 *      Otherwise a new element is created.
 *
 *  \image html resources/edit_delete.png
 *  \image latex resources/edit_delete.png "" width=3cm
 *  \li Deletes all pink-cornered elements. This operation may fail if marked elements have unmarked children.
 *
 *  \image html resources/edit_instantiate.png
 *  \image latex resources/edit_instantiate.png "" width=3cm
 *  \li Toggles the pink-cornered classifiers between classes and instances. (Does not work for relationships and features)
 *
 *  \image html resources/edit_highlight.png
 *  \image latex resources/edit_highlight.png "" width=3cm
 *  \li Toggles the pink-cornered classifiers between yellow-marked, greyed-out and normal. (Does not work for relationships and features)
 *
 *  \image html resources/edit_reset.png
 *  \image latex resources/edit_reset.png "" width=3cm
 *  \li Resets the pink-cornered selection
 *
 *  \image html resources/edit_undo.png
 *  \image latex resources/edit_undo.png "" width=3cm
 *  \li Un-does the last operation. (Opening a database, exporting files cannot be undone)
 *
 *  \image html resources/edit_redo.png
 *  \image latex resources/edit_redo.png "" width=3cm
 *  \li Re-does the last un-done operation.
 *
 *  \image html resources/crystal_facet_uml.png
 *  \image latex resources/crystal_facet_uml.png "" width=3cm
 *  \li Shows version, license and copyrights
 *
 *  \image html resources/edit_commit.png
 *  \image latex resources/edit_commit.png "" width=3cm
 *  \li Stores the latest changes to the database (as most other displayed widgets too). It is not necessary to explicitly save the file.
 *
 *  \image html resources/message_info.png
 *  \image latex resources/message_info.png "" width=3cm
 *  \li Informs on success of an operation, e.g. an export
 *
 *  \image html resources/message_warn.png
 *  \image latex resources/message_warn.png "" width=3cm
 *  \li Informs on a possible problem
 *
 *  \image html resources/message_error.png
 *  \image latex resources/message_error.png "" width=3cm
 *  \li Informs on an error
 *
